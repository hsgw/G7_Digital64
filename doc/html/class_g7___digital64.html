説明

4つのI2C IO Expander ICをコントロールして64本のデジタルIOを追加します

覚え書き
このシールドを2枚使用したいときは基板上のADDR_SELで選択するアドレスをそれぞれ変えること
IC番号には DIO_A / DIO_B / DIO_C / DIO_D が使用できます。
